agentforce-md

Convert Claude Code markdown to Salesforce Agentforce Agent Script

Install

curl -sSL https://agentforce-md.herokuapp.com/install.sh | bash

Prerequisites

  • Python 3.10+ — python3 --version
  • Claude Code installed — ~/.claude/ must exist
  • Salesforce CLI — optional — sf --version

Skills

/agentforce-convert

Full round-trip: markdown → Agent Script → deploy to your org

/agentforce-discover

Check which SKILL.md targets (Flows, Apex, Retrievers) exist in your org

/agentforce-scaffold

Generate Flow / Apex stubs for targets that are missing from the org

/agentforce-run

Execute a SKILL.md action against a live org via REST API

Usage after install

# Open Claude Code in any project and type:
/agentforce-convert

# Or run the CLI directly:
~/.claude/agentforce-md/bin/agentforce-md convert \
  --project-root . --agent-name MyAgent
